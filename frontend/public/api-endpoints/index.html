API REST Endpoints

Integra el motor de proteostasis en tus propias aplicaciones o análisis de datos.

Consola Interactiva (Swagger)
GET

/api/patient/{patient_id}

Obtiene el perfil biológico, genético y biomarcadores de un paciente específico de la base de datos.

Respuesta Exitosa (200 OK)
{
  "id": "1",
  "name": "María González",
  "age": 65,
  "genotype": { "APOE": "ε4/ε4" },
  "p_tau217": 3.8
}
POST

/simulate

Ejecuta el motor de SciPy para calcular la progresión espaciotemporal de Tau y Aβ en base al conectoma cerebral.

Cuerpo de la Petición (JSON)
{
  "patient": { ... },
  "interventions": {
    "anti_Aβ": 1.0,
    "anti_tau": 0.5
  },
  "duration_days": 365
}
POST

/optimize

Ejecuta el algoritmo genético NSGA-III para buscar combinaciones terapéuticas en el Frente de Pareto.
