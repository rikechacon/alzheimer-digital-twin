## 🚀 Cómo Contribuir

### 1. Elige tu área de contribución

- **🧪 Validación científica**: Comparación con cohortes públicas (ADNI, BioFINDER)
- **💻 Desarrollo de código**: Simulador, optimizador, API, frontend
- **📊 Visualización**: Dashboard clínico, gráficos interactivos
- **🌐 Internacionalización**: Traducción a múltiples idiomas
- **📱 Mobile**: App para pacientes con monitoreo de adherencia
- **🤖 Machine Learning**: Mejora de surrogate models con transformers
- **📝 Documentación**: Tutoriales, guías, papers técnicos

### 2. Sigue el flujo de contribución

```bash
# 1. Fork el repositorio
# Haz clic en "Fork" en la esquina superior derecha de GitHub

# 2. Clona tu fork
git clone https://github.com/tu-usuario/alzdt-core.git
cd alzdt-core

# 3. Crea una rama para tu feature
git checkout -b feature/nombre-de-tu-feature

# 4. Haz tus cambios y commit
git add .
git commit -m "feat: descripción de tu cambio"

# 5. Push a tu rama
git push origin feature/nombre-de-tu-feature

# 6. Abre un Pull Request en GitHub
# Ve a tu fork → Pull Requests → New Pull Request

# 🤝 Código de Conducta del Alzheimer Digital Twin

## Nuestro Compromiso

En el proyecto Alzheimer Digital Twin, estamos comprometidos a crear un entorno abierto, inclusivo y respetuoso para todos los contribuidores, independientemente de su edad, cuerpo, discapacidad, etnia, identidad de género y expresión, nivel de experiencia, nacionalidad, apariencia física, raza, religión u orientación sexual.

## Nuestros Estándares

### Comportamientos que fomentamos:
- **Respeto mutuo**: Tratar a todos con dignidad y consideración
- **Colaboración abierta**: Compartir conocimientos y ayudar a otros
- **Crítica constructiva**: Dar y recibir feedback de manera respetuosa
- **Responsabilidad**: Aceptar responsabilidad por tus acciones y errores
- **Inclusividad**: Asegurar que todos se sientan bienvenidos y valorados

### Comportamientos inaceptables:
- **Acoso**: Comentarios ofensivos, insultos, ataques personales
- **Discriminación**: Exclusión basada en características personales
- **Intimidación**: Comportamiento amenazante o agresivo
- **Falta de respeto**: Desestimar las contribuciones de otros
- **Comportamiento no profesional**: Lenguaje inapropiado, spam, trolling

## Responsabilidades de Aplicación

Los mantenedores del proyecto son responsables de:
- Clarificar los estándares de comportamiento aceptable
- Tomar acciones correctivas apropiadas ante comportamientos inaceptables
- Eliminar, editar o rechazar comentarios, commits, código, issues y PRs que no se alineen con este Código de Conducta
- Prohibir temporal o permanentemente a contribuidores por comportamientos inapropiados

## Alcance

Este Código de Conducta aplica a:
- Todos los espacios del proyecto (GitHub, Discord, reuniones, etc.)
- Comunicaciones públicas relacionadas con el proyecto
- Eventos oficiales del proyecto (conferencias, meetups, etc.)

## Reportar Problemas

Si experimentas o presencias comportamiento inaceptable:

1. **Contacta al equipo**: alzdt.ethics@digitaltwin.org
2. **Proporciona detalles**: Qué sucedió, cuándo, quiénes estuvieron involucrados
3. **Mantén confidencialidad**: No compartas públicamente detalles sensibles

Todas las quejas serán revisadas y respondidas de manera justa y oportuna.

## Procedimiento de Aplicación

1. **Recepción**: El equipo recibe y documenta el reporte
2. **Investigación**: Se investiga el incidente de manera imparcial
3. **Decisión**: Se determina la acción apropiada basada en la gravedad
4. **Acción**: Se implementa la acción correctiva
5. **Seguimiento**: Se monitorea la situación para prevenir recurrencia

## Consecuencias de Comportamiento Inaceptable

Las acciones correctivas pueden incluir:
- **Advertencia privada**: Conversación educativa sobre el comportamiento
- **Advertencia pública**: Notificación pública en el canal relevante
- **Suspensión temporal**: Pérdida temporal de privilegios o acceso
- **Prohibición permanente**: Eliminación del proyecto y comunidad

## Atribución

Este Código de Conducta está adaptado del [Contributor Covenant](https://www.contributor-covenant.org/), versión 2.1, disponible en https://www.contributor-covenant.org/version/2/1/code_of_conduct.html
